# 🛠 WBCreator v1.5 — редактор карточек Wildberries

Браузерный инструмент для создания карточек товаров WB с удалением фона через AI.

## 🚀 Как опубликовать на GitHub Pages (бесплатный хостинг)

### Шаг 1 — Создай репозиторий
1. Зайди на [github.com](https://github.com) → войди в аккаунт (или зарегистрируйся)
2. Нажми зелёную кнопку **New** (или **+** → **New repository**)
3. Название репозитория: `wb-creator` (или любое другое)
4. Видимость: **Public** ✅ (обязательно, иначе Pages не работает бесплатно)
5. Нажми **Create repository**

### Шаг 2 — Загрузи файл
1. На странице репозитория нажми **uploading an existing file** (или **Add file → Upload files**)
2. Перетащи файл `index.html` в окно загрузки
3. Также загрузи этот `README.md` (опционально)
4. Внизу нажми **Commit changes**

### Шаг 3 — Включи GitHub Pages
1. Перейди в **Settings** репозитория (вкладка сверху)
2. В левом меню найди раздел **Pages**
3. В разделе **Branch** выбери `main` и папку `/ (root)`
4. Нажми **Save**

### Шаг 4 — Открой сайт
Через 1–2 минуты твой сайт будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/wb-creator/
```
Этот адрес можно давать клиентам!

## 🔧 Функции приложения

| Кнопка | Что делает |
|--------|------------|
| 📷 Загрузить товар | Загружает фото с компьютера |
| ✂️ Удалить фон (AI) | Убирает фон через remove.bg API |
| 🟩 Добавить плашку | Жёлтый прямоугольник-бейдж |
| ✍️ Добавить текст | Редактируемый текстовый блок |
| 🔼 Вынести вперёд | Поднимает объект поверх остальных |
| 📐 Применить шаблон | Готовый макет карточки WB |
| ⬇️ Скачать (HD) | Сохраняет карточку 900×1200px JPEG |

**Удаление объекта:** выдели → нажми `Delete`

## 🔑 Получить ключ remove.bg

1. Зайди на [remove.bg](https://www.remove.bg/ru)
2. Зарегистрируйся (бесплатно — 50 удалений в месяц)
3. Profile → API Keys → Create new key
4. Вставь ключ в поле приложения (сохраняется автоматически)
